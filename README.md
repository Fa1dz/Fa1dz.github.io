## 🚀 About

Welcome to my personal developer portfolio.

This website automatically connects to the GitHub API and displays my public repositories, projects, and development activity.

The design focuses on:

- Modern UI
- Smooth animations
- Performance
- Clean code
- Cybersecurity-inspired aesthetics

## ✨ Features

### 🎨 Design

- Glassmorphism interface
- Animated gradient background
- Interactive particle network
- Custom cursor effects
- Responsive layout
- Smooth scrolling


### 🐙 GitHub Integration

- Automatic repository loading
- Live repository data
- Stars and forks
- Programming language detection
- Repository search


### ⚡ Performance

- No frameworks
- No dependencies
- Lightweight JavaScript
- GitHub Pages compatible
